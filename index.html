Web Development is the building and maintenance of websites; it's the work that happens behind the scenes to make a website look great, work fast and perform well with a seamless user experience. web developers, or 'devs', do this by using a variety of coding languages

html free course

(HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

css free course

(CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML).[1] CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

Bootstrap free course

Bootstrap is a free, open-source front-end development framework that helps developers and designers build websites and web apps. It was originally created by a Twitter designer and developer in 2010 and was previously called Twitter Blueprint. Bootstrap is maintained by a small team of developers on GitHub.
